# alias 환경변수 설정방법

* alias는 지역변수

vim ~/.bashrc_alias
에 원하는 환경변수 적기
```bash
alias grep=`grep --color-auto
````

~/.bashrc
```bash
if [ -f ~/.bash_alias]; then
    . ~/.bash_alias
fi
```
