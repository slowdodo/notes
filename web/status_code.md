 |                   응답					| 상태 코드<br>(상세 코드)	|        응답 구문			|                                                    설명<br>														|
|:---------------------------------------:	|:------------------------:	|:---------------------:	|:---------------------------------------------------------------------------------------------------------------:	|
|        1xx: Information<br>(정보)			|            100			| Continue					| 클라이언트로부터 일부 요청을 받았으며 나머지 정보를 계속 요청														|
|											|            101			| Switching					| 서버는 Upgrade 헤더에 정의된 프로토콜을 변환하기 위해 클라이언트의 요청을 수락									|
|         2xx : Success<br>(성공)			|            200			| OK						| 요청이 성공적으로 수행됨																							|
|											|            201			| Created					| PUT 메소드에 의하여 원격 서버에 파일 생성																			|
|											|            202			| Accepted					| 웹 서버가 명령을 수신함																							|
|											|            204			| No Content				| 요청을 정상적으로 처리하였지만, 클라이언트에 보낼 헤더 또는 바디가 없음. 웹브라우저는 기존과 동일한 페이지 유지	|
|											|            206			| Partial Content			| Range가 지정되었을 때, 그 범위만큼 요청을 받았다는 것을 의미. 응답 시 Content-Range 포함							|
|     3xx : Redirection<br>(리디렉션)		|            301			| Moved Permanently			| 요청 자원의 위치가 영구적으로 변경. 변경 위치는 Location으로 전달													|
|											|            302			| Found						| 요청 자원의 위치가 임시적으로 변경																				|
|											|            304			| Not Modified				| 요청한 자원에 변경이 없으므로 클라이언트 로컬 캐시에 저장된 자원 이용												|
|											|            307			| Temporary Redirect		| 302와 동일한 의미를 가지고 있지만, 클라이언트의 HTTP 메소드를 변경하지 말아야 함									|
|											|            308			| Permanent Redirect		| 301과 동일한 의미를 가지고 있지만, 클라이언트의 HTTP 메소드를 변경하지 말아야 함									|
| 4xx : Client Error<br>(클라이언트 오류)	|            400			| Bad Request				| 요청 메시지 문법 오류																								|
|											|            401			| Unauthorized				| 요청한 자원에 적절한 권한이 없음																					|
|											|            403			| Forbidden					| 요청한 자원에 접근 차단																							|
|											|            404			| Not Found					| 요청한 자원이 존재하지 않음																						|
|											| 			 405			| Method not allow			| 클라이언트가 서버에 요청한 자료가 존재하지 않음을 나타냄															|
|											| 			 406			| Not acceptable			| 클라이언트 요청에 이용한 메소드가 해당 URL에 지원이 불가능하다는 것을 나타냄										|
|    5xx : Server Error<br>(서버 오류)		|            500			| Internal Server Error		| 내부 서버 오류																									|
|											|            501			| Not Implemented			| 요청 방법이 서버에서 지원되지 않아서 요청을 처리하지 못함															|
|											|            502			| Bad Gateway				| 게이트웨이가 프로토콜에 문제가 있거나 잘못된 프로토콜을 통해 연결하여 통신이 정상적이지 않음						|
|											|            503			| Service Unavailable		| 서버가 요청을 처리할 준비가 되지 않음																				|
