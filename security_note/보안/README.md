보안의 3대 요소

    1. 기밀성(비밀성, Confidentiality)
        인가된 사용자만이 정보자산에 접근할 수 있도록 하는것

    2. 무결성(Integrity)
        적절한 권한을 가진 사용자가 인가된 방법으로만 정보를 변경할 수 있도록 하는것

    3. 가용성(Availability)
        정보자산에 대한 필요한 시점에 접근이 가능하도록 하는 것

AAA(Triple-A)

    1.인증(Authenticaltion)

    2.권한부여(Authorization)

    3.계정관리(Accounting)
