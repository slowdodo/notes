
# 인증제도의 특징
* 국내 실정에 적합한 정보보호관리 모델 제시
* 공신력 있는 정보보호 전문기관(KISA)에 의한 심사 및 인증
* 국내 최고의 분야별 전문가들에 의한 인증 심사
* 국내 정보보호관련 법제도 반영
