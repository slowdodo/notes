# TP(Transparent) Mode or Bridge Mode
* 방화벽이 스위치(혹은 브릿지) 처럼 작동하며 패킷 필터를 한다
* 인터페이스가 2개 있다.
* 2개의 인터페이스는 같은 네트워크 안에 있다. (subnet이 같음)
* 맥 어드레스를 기반으로 패킷이 어느 방향에서 왔는지를 판단한다.
* 내부 및 외부 인터페이스가 같은 네트워크 안에 있음(브리지 구성)
* 구축 시 기존 네트워크 환경을 변경할 필요가 없음
