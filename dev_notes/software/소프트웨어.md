# 소프트웨어 특징
* 논리성 : 형상이 없고 논리적인 것으로 존재
* 개발성(비제조성) : 물리적 제조 또는 생산이 아닌 논리적으로 짬
* 비마모성 : 오랜 기간 사용해도 마모되지 않음
* 유연성(순응성) : 사용자 요구나 환경에 적절히 언제든지 수정이 가능
* 유형성 : 원시 프로그램(코드)나 문서는 시각적임
* 상품성 : 소프트웨어는 판매가 가능
* 견고성 : 일부 수정으로 전체에 영향을 줄 수 있음 (-> 구조가 견고해야 함)
