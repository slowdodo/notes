|     항목     |                                            설명                                            |
|:------------:|:------------------------------------------------------------------------------------------:|
| Destination  | 목적지 네트워크                                                                            |
|    Gateway   | 목적지로 가기위한 게이트웨이 주소                                                          |
|    Genmask   | 목적지 네트워크의 넷마스크 주소 (목적지IP와 Genmask를 AND 연산한 결과가 목적지 네트워크임) |
|     Flags    | 해당 경로에 대한 정보를 알려주는 기호/ U - up, H - 목적지 Host, G - 게이트웨이 사용        |
|    Metric    | 목적지 네트워크까지의 거리                                                                 |
|      Ref     | 경로를 참조한 횟수                                                                         |
|      Use     | 경로를 탐색한 횟수                                                                         |
|     IFace    | 네트워크 인터페이스                                                                        |
