# Echo란?
기본적으로 리눅스, 유닉스, 프리도스, 윈도우 대부분 이 기능을 지원하는데

화면에 텍스트 출력을 도와주는 기능이다. 매우 단순하지만 매우 유용하게 사용도 가능한 기능이다.

# 화면 출력

> echo hello
``` bash
hello
```
# 리눅스에서만

``` bash
echo {A..Z}
```

``` bash
echo {1..200}
```

# example

``` bash
echo google.com > test.txt 
```

``` bash
nslookup `echo test.txt`
```
