# 자료구조의 개념

    자료를 효율적으로 표현하고 저장하고 처리할수 있도록 정리하는 것

# 자료의 형태에 따른 분류

단순 구조

    정수, 실수, 문자, 문자열 등 기본 자료형

선형 구조

    자료들 사이에 관계가 1:1 관계
    ex) 순차 리스트, 연결 리스트, 스택, 큐, 데크 등

비선형 구조

    자료들 사이의 관계가 1:다 또는 다:다

    ex) 트리, 그래프

파일 구조

    서로 관련 있는 필드로 구성된 레코드의 집하인 파일에 대한 구조
    ex)순차 파일, 색인 파일, 직접 파일 등
